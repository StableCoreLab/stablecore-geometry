# Box 设计

## 1. 目的

本文定义当前仓库中 2D `Box2<T>` 体系的设计边界，作为后续维护、文档示例和 API 收口的参考基线。

## 2. 核心语义

`Box2<T>` 是包围盒对象，用于表达最小轴对齐边界范围。

## 3. 天然成员能力

适合保留为成员方法的包括：

- `MinPoint()`
- `MaxPoint()`
- `Bounds()`
- `IsValid()`
- `Expand(...)`

## 4. 边界约束

`Box2<T>` 不应承担：

- 复杂空间索引逻辑
- 多对象布尔流程
- 高层业务语义

## 5. 当前固定结论

- box 保持轻量值类型
- API 命名使用短名风格
- 仅提供边界对象天然成立的能力
